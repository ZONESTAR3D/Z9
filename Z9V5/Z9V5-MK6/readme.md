## Firmware for Z9V5Pro-MK6
**:pencil: What is firmware bin file and source code?**    
Firmware bin file is the exact memory that is written to the embedded flash. Firmware source code is the core part of the firmware, the firmware source code of Z9V5 is base on marlin, for more about marlin firmware, please refer to [**:point_right:marlinfw.org**](https://www.marlinfw.org).    
You can download firmware bin file or source code from the below link:
- **[:arrow_down: Download firmware bin file](https://github.com/ZONESTAR3D/Firmware/tree/master/Z9/Z9V5/bin/Z9V5Pro-MK6)**   
- **[:arrow_down: Download firmware source code](https://github.com/ZONESTAR3D/source-code-for-3d-printer)**
